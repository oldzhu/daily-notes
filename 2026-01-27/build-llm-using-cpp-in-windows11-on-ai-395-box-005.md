### 第二部分：梯度是如何计算的？（数学推导）

这个计算过程是微积分中**链式法则** 的一个直接应用。

为了让公式更简洁，我们先定义一些术语：
*   令损失函数为 `L = (wx + b - target)^2`
*   令 `error`（误差）为 `e = (wx + b - target)`

所以，我们的损失函数可以简单地写成 `L = e^2`。

梯度是一个由**偏导数** 组成的向量。我们需要分别计算损失函数 `L` 对 `w` 的偏导数，以及 `L` 对 `b` 的偏导数。

#### 1. 计算关于 w 的梯度 (∂L/∂w)

我们想要求解 `∂/∂w (e^2)`。使用链式法则：
`d/dw [f(g(w))] = f'(g(w)) * g'(w)`

*   我们的**外层函数**是 `f(e) = e^2`。它的导数是 `f'(e) = 2e`。
*   我们的**内层函数**是 `g(w) = wx + b - target`。它对 `w` 的导数是 `g'(w) = x`。（记住，当我们对 `w` 求导时，`b`、`x` 和 `target` 被视为常数）。

现在，我们将它们相乘：
`∂L/∂w = (2e) * (x)`
`∂L/∂w = 2 * (wx + b - target) * x`
`∂L/∂w = 2x(error)`

**这与你的第一个陈述完全匹配！**

#### 2. 计算关于 b 的梯度 (∂L/∂b)

我们想要求解 `∂/∂b (e^2)`。我们再次使用链式法则。

*   **外层函数**仍然是 `f(e) = e^2`。它的导数是 `f'(e) = 2e`。
*   **内层函数**是 `g(b) = wx + b - target`。它对 `b` 的导数是 `g'(b) = 1`。（现在，`w`、`x` 和 `target` 被视为常数）。

现在，我们将它们相乘：
`∂L/∂b = (2e) * (1)`
`∂L/∂b = 2 * (wx + b - target)`
`∂L/∂b = 2(error)`

**这与你的第二个陈述完全匹配！**
